# Cursor MCP 설정 가이드

## 현재 상태
MCP 설정 파일은 `~/.cursor/mcp.json`에 이미 존재합니다.

## 프로젝트 ID 확인
현재 프로젝트의 Supabase Project ID: `gcpdcotbvsvyuthqhxoi`

## 설정 방법

### 1. Supabase Personal Access Token 생성
1. https://supabase.com/dashboard 접속
2. 오른쪽 상단 프로필 → "Account Settings"
3. "Access Tokens" 메뉴 선택
4. "Generate New Token" 클릭
5. 이름 입력 (예: "Cursor MCP") 후 생성
6. 토큰 복사 (한 번만 표시됨!)

### 2. MCP 설정 파일 수정

`~/.cursor/mcp.json` 파일을 열고 다음 내용으로 수정하세요:

```json
{
  "mcpServers": {
    "supabase": {
      "command": "npx",
      "args": [
        "-y",
        "@supabase/mcp-server-supabase@latest",
        "--project-ref=gcpdcotbvsvyuthqhxoi"
      ],
      "env": {
        "SUPABASE_ACCESS_TOKEN": "여기에_생성한_토큰_붙여넣기"
      }
    }
  }
}
```

**중요:**
- `--project-ref=gcpdcotbvsvyuthqhxoi` - 현재 프로젝트 ID로 변경
- `SUPABASE_ACCESS_TOKEN` - 새로 생성한 토큰으로 교체

### 3. Cursor 재시작
설정을 적용하려면 Cursor를 완전히 종료한 후 다시 실행하세요.

### 4. 연결 확인
재시작 후 MCP 도구가 작동하는지 확인:
- 데이터베이스 테이블 조회 가능
- 마이그레이션 실행 가능
- SQL 쿼리 실행 가능

## 보안 주의사항
⚠️ **절대 공개하지 마세요:**
- `SUPABASE_ACCESS_TOKEN`은 절대 GitHub나 공개 저장소에 커밋하지 마세요
- 토큰이 노출되면 즉시 재생성하세요
- 프로젝트 ID는 공개되어도 괜찮지만, 토큰은 비밀로 관리하세요
